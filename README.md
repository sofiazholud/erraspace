# Erra Space Studio Website

Static website for **Erra Space Studio** (Вишгород та Міжріччя, Україна), hosted on GitHub Pages.

## Live site

https://erraspace.com.ua

## Structure

- `index.html` — головна сторінка (квіз, напрямки, відгуки, FAQ, подорожі)
- `prices.html` — абонементи та ціни
- `classes.html` — детальний опис напрямків
- `trainers.html` — сторінка тренерів
- `locations.html` — адреси студій та контакти
- `about.html` — про студію, місія та цінності
- `travel.html` — виїзні ретрити та подорожі
- `cappadocia.html` — програма ретриту в Каппадокії
- `oferta.html` — публічна оферта
- `style.css` — стилі
- `script.js` — логіка перемикання мов, модальні вікна, FAQ
- `images/` — зображення для напрямків та тренерів

## Features

- Двомовність (UA/EN)
- Інтерактивний квіз для підбору напрямку
- Модальні вікна для опису напрямків
- Адаптивна сітка для карток
- Сторінка-презентація ретриту в Каппадокії
- Google Analytics (GA4) інтеграція

## Технології

- HTML5 / CSS3 / JavaScript
- GitHub Pages для хостингу
- Lazy loading для оптимізації зображень
- Sitemap.xml + robots.txt для SEO

## SEO та індексація

- Додано `sitemap.xml` для швидкої індексації
- Додано `robots.txt` для відкритої індексації
- Підтвердження домену у Google Search Console (через TXT-запис або мета-тег)

## Deployment

Сайт розгорнуто через **GitHub Pages**:

```
git clone https://github.com/sofiazh/erraspace.git
cd erraspace
```

Зміни в гілці `main` автоматично публікуються на сайт.
